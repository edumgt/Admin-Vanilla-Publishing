Basic Accordion

This is the first accordion item. Each accordion contains accordion-item. accordion-item contains accordion-header and accordion-body. accordion-header contains accordion-button. accordion-item can be opened by adding open class to it. no-collapse class will prevent accordion-item from opening. Add parent attribute to each accordion-item so that only one accordion-item is opened at a time. If you omit the parent attribute the accordion-item will always remain opened. Add no-arrow class to accordion-item to hide the arrow.
